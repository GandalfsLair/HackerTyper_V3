A standard hackertyper with hotkeys assigned to single press of 1 - 0

Pressing RETURN/ENTER triggers autoscroll.

Pressing RETURN/ENTER again turns autoscroll off.

When autoscroll is off, mashing the keyboard will give characters to screen from the code.txt file at a rate of one character per button press. 

Hotkeys function regardless of whether autoscroll is on or off.
 
For best results, run in MS Studio Code via Live Server Extension
